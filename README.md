# 🎓 Grade Calculator

## 📌 Description
This is my first JavaScript mini project.
It calculates a student's grade based on the marks entered by the user.

## 🚀 Features
-✅Takes marks as input using prompt()
-✅Validates input (0-100)
-✅Displays Grade (A,B,C,D,F)
-✅Shows Pass/Fail result
-✅Displays a performance message

## 🛠️ Technologies Used 
-HTML5
-JavaScript

## How to Run
1. Open index.html in a browser.
2. Enter your marks.
3. Check the result in the browser console.

## 📚 Concepts Used
-Variables
-if-else
-Comparison Operators
-Logical Operators
-Prompt()
-console.log()
-Input Validation
